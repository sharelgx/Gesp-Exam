📐 找出勾股三元组（a² + b² = c²）

#include <iostream> #include <cmath> using namespace std; for (int a = 1; a < 50; ++a) { for (int b = a; b < 50; ++b) { double c = sqrt(a * a + b * b); if (c == (int)c) { cout << a << "^2 + " << b << "^2 = " << (int)c << "^2" << endl; } } }
📚 我们将逐个尝试 a 和 b，看看是否能找到一个整数 c，使得 a² + b² = c²！
